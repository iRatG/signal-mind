# Signal Mind — Контекст для модели

## Как использовать эти файлы

Ты получил эту папку чтобы понять проект Signal Mind и продолжить работу.
Читай файлы в порядке нумерации:

1. `01_idea.md` — что мы строим и зачем
2. `02_data_map.md` — полная карта данных: что есть, где лежит, какой формат
3. `03_tech_stack.md` — технические решения: какие БД, RAG, LLM, почему именно они
4. `04_action_plan.md` — пошаговый план реализации по фазам
5. `05_parsing_notes.md` — как парсить каждый тип файлов (нюансы и подводные камни)

## Важно о стиле работы

- Пользователь хочет **сначала думать вместе**, потом делать
- Не прыгай сразу к коду без подтверждения
- Покажи общую картину словами, получи "ок" — потом реализуй
- Объясняй как человеку который понимает бизнес, но не обязан знать детали кода
